Importante

O GitHub Pages hospeda apenas conteudo estatico. A aplicacao completa continua dependendo do backend Express e do MongoDB para funcionar.

O que o projeto cobre

  • Gestao comercial Clientes, pedidos, orcamentos e acompanhamento do fluxo de producao.
  • Controle operacional Materiais, equipamentos, compras, configuracoes e catalogos auxiliares.
  • Base para expansao Estrutura pronta para crescer com autenticacao, planos e recursos administrativos.

Stack atual

  • Frontend HTML, CSS e JavaScript servidos pelo backend.
  • Backend Node.js com Express e organizacao por controllers, services e routes.
  • Banco MongoDB com Mongoose.

Publicacao web

Esta pagina estatica foi preparada para ser publicada pelo GitHub Pages. Ela funciona como uma vitrine do projeto no navegador, enquanto a aplicacao principal continua rodando localmente ou em uma hospedagem com suporte a Node.js e MongoDB.

Proximos passos para a app completa

  • Hospedagem do backend Subir o servidor Express em um provedor com suporte a Node.js.
  • Banco gerenciado Apontar `MONGO_URI` para uma instancia publica ou privada do MongoDB.
  • Frontend integrado Ajustar variaveis e endpoints para consumir a API publicada.
